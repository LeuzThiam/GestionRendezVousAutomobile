# Frontend React

Application frontend du projet de gestion de rendez-vous automobile.

## Demarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuration API

Par defaut, le frontend appelle:

```bash
http://127.0.0.1:8000
```

Pour changer l'URL, definir la variable Vite suivante:

```bash
VITE_API_URL=http://127.0.0.1:8000
```
